
安装软件
1. 下载***.tar.gz文件
2. 解压 tar zxf ***。tar.gz
3. cd 解压目录
4. 设置安装配置参数，如安装路径 ./configure PREFIX=/opt/*** 
5. 编译 make
6. 安装 make install

添加忽略文件并删除远程对应文件
1. 移除git管理
 git rm -r --cached 特定文件		（**/application.properties）
2. 在 .gitignore中添加忽略
3.更新git管理
git add .
4. 提价修改到本地
git commit -m '修改忽略文件'
5. 推送到远程仓库
git push original master

springboot启动脚本
1.创建对应启动脚本restart.sh
	restart.sh
		source ~/.bash_profile
		killall java
		nohup java -jar /root/springbootdemo/springbootdemo-0.0.1-SNAPSHOT.jar > nohup.log 2>&1 &
2.执行启动脚本
	sh /root/springbootdemo/restart.sh
